A question and answers Portal

Install

Just Clone the project code from github . Paste the contents of the folder in htdocs folder after setting up Apache Web Server ( Or anmy Web server for that matter ) .Then create a database "qa" in Mysql and then import the MYSQL.qa.txt file from the code.
